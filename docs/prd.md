# 车辆通行费票据识别小程序需求文档

## 1. 小程序概述
### 1.1 小程序名称
智票通

### 1.2 小程序描述
专门用于识别车辆通行费票据信息的微信小程序，通过拍照或上传图片自动提取票据中的关键信息，提高缴费记录管理和报销效率。

## 2. 核心功能
### 2.1 拍照上传功能
- 支持调用手机摄像头拍照
- 支持从相册选择图片上传
- 图片预处理（裁剪、增强清晰度）

### 2.2 OCR识别功能
- 车牌号码识别：自动提取票据中的车牌号信息
- 通行费用识别：准确识别缴费金额
- 时间信息提取：提取通行时间和缴费时间
- 路段信息识别：识别起点、终点和途经省份
- 车辆信息：车型、轴数等

### 2.3 识别结果展示
- 清晰展示提取的所有关键信息
- 支持结果编辑和修正
- 一键保存识别记录
- 支持导出为Excel或PDF格式

### 2.4 历史记录
- 保存历次识别结果
- 支持按时间、车牌号筛选
- 批量删除功能

## 3. 参考素材
### 3.1 标记图片
使用用户提供的图片蓝牌标记.png作为OCR识别的参考模板，该图片已清晰标记了需要识别的各个区域位置。

## 4. 设计风格
### 4.1 配色方案
主色调采用蓝色系（#1890FF），辅以白色背景和灰色文字，体现专业可靠的科技感

### 4.2 视觉细节
界面采用卡片式布局，圆角设计增加亲和力，识别结果使用表格形式清晰展示各项数据

### 4.3 交互体验
拍照界面采用悬浮按钮设计，识别结果页面支持滑动刷新，整体操作流畅便捷
